## Format des images

- **Format recommandé** : PNG avec transparence
- **Taille recommandée** : 200x200 pixels minimum (carré)
- **Nommage** : Tout en minuscules, sans accents, avec tirets pour les espaces
- **Exemple** : `professeur-chen.png` (pas `Professeur Chen.png` ou `professeur_chen.png`)

## Notes importantes

⚠️ **Attention à la casse** : Sur GitHub Pages, `Chen.png` ≠ `chen.png`
⚠️ **Pas d'espaces** : Utilisez des tirets `-` au lieu d'espaces
⚠️ **Pas d'accents** : Évitez les caractères accentués dans les noms de fichiers

## Test

Pour tester si une image est accessible, ouvrez directement l'URL dans votre navigateur :
```
https://votre-username.github.io/votre-repo/assets/img/dresseurs/professeur-chen.png
```

Si vous voyez l'image, le chemin est correct. Si vous obtenez une erreur 404, vérifiez :
- Le nom du fichier (casse exacte)
- L'emplacement du fichier
- L'extension du fichier (.png, .jpg, etc.)
